# DomiMD
DomiMD - Markdown Editor
----
Markdown editor tool with customisation for:
- font size in em:
- + 3.0em {3em:test}
- + 2.5em {2.5em:test}
- + 2.0em {2em:test}
- + 1.5em {1.5em:test}
- + normal test
- + 0.75em {0.75em:test}

- font colour:
- + {RED:Red},{BLU:Blue}, {YEL:Yellow}
- + {GRN:Green}, {ORG:Orange}, {CYN:Cyan}
- + {LME:Lime}, {GRY:Gainsboro}, {PNK:Pink}
- + White,Black
- background colours:
- + {RED.B:Red},{BLU.B:Blue}, {YEL.B:Yellow}
- + {GRN.B:Green}, {ORG.B:Orange}, {CYN.B:Cyan}
- + {LME.B:Lime}, {GRY.B:Gainsboro}, {PNK.B:Pink}
- + {BLK.B:{WHT:Black}},{SMK.B:Whitesmoke}
- can be combined:
- + {BLU.B:{YEL:such as}} in {GRN.B:{LME:this}} example
